# TicketFlow Backend Service

Built a production-style backend service using FastAPI and PostgreSQL to simulate ticket lifecycle tracking and operational analytics.

Highlights:

- Designed PostgreSQL schemas supporting ~1M ticket lifecycle events
- Implemented indexing and partitioning strategies for performance optimization
- Added async database access using asyncpg and connection pooling
- Developed analytics endpoints and aggregation workflows
- Deployed backend services using Nginx and systemd on Ubuntu Server
- Added CI/CD workflows, logging, observability, and testing
